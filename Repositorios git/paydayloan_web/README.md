**PAYDAYLOAN**
---------------------------
Rails 4 implementation Bootstrap admin template.

Setup:
 1. Clone this repo :  git clone    

 2. cd paydayloan_web

 3. bundle install

 4. rails s

Navigate to   http://localhost:3000
